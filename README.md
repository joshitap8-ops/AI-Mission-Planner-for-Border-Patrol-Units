# AI Mission Planner for Border Patrol Units

AI Mission Planner for Border Patrol Units is a defence-focused intelligent mission planning platform designed to assist border security teams in route optimization, threat prediction, drone coordination, and real-time tactical decision-making.

The system uses **AI-powered path planning algorithms (A*)**, **terrain-aware risk scoring**, and **simulation-based intrusion response workflows** to generate the safest and most efficient patrol routes.

It provides a **military-style command center dashboard** with:

* live tactical map visualization
* danger zone detection
* patrol route optimization
* drone station management
* intrusion hotspot simulation
* route rerouting during emergencies
* mission analytics and replay

The platform is built for **hackathon-scale defence software demonstrations**, combining:

* **React + Tailwind frontend**
* **Leaflet tactical map**
* **FastAPI backend**
* **AI/ML risk scoring**
* **CSV / GeoJSON based terrain datasets**

This project demonstrates how AI can improve **border surveillance, patrol efficiency, resource utilization, and rapid response coordination** in high-risk defence environments.
